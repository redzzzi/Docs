## 19.2 Simple PHP Program
```PHP
<!DOCTYPE html>
<html>
<?php
  $name = "Paul";
?>
  <head>
    <meta charset = "utf-8">
    <title>Simple PHP document</title>
  </head>
  <body>
    <h1><?php print("Welcome to PHP, $name!"); ?></h1>
  </body>
</html>
```
- php 태그(`<?php` `?>` 내부에 `$name`이라는 변수를 선언
- 이후 h1 태그 내부에서 해당 변수를 출력
